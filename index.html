Perguntas Frequentes

Como funciona o sistema de denúncias?

Você preenche um formulário com informações sobre o incidente. O sistema valida os dados e envia para análise pelos órgãos competentes. Você receberá um número de protocolo para acompanhar sua denúncia.

Minha denúncia é realmente anônima?

Sim. Se você optar pela denúncia anônima, nenhuma informação pessoal será registrada ou transmitida. Seus dados estão protegidos por criptografia de ponta a ponta.

Quanto tempo demora para investigar?

O tempo varia conforme a complexidade do caso. Crimes graves são investigados com prioridade. Você poderá acompanhar o progresso usando seu número de protocolo.

Posso anexar provas?

Sim, você pode enviar fotos, vídeos e documentos (até 50MB). Estes arquivos são seguros e criptografados.

É uma emergência?

Para emergências, ligue 190 imediatamente. Este sistema é para denúncias que não exigem resposta imediata.
